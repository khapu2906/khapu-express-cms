## Install
```

npm i spy-84-cms

```

---
## Usage

* Generate structure by:

  ```
  spy@prepare

  ```
    **or**
  ```
  npx spy@prepare

  ```

* Run by
  ```
  spy@go

  ```
    **or**
  ```
  npx spy@go

  ```
  **or**
  ```
  node server.js

  ```
